# Shopping Cart

This project is designed to be a fully functional shopping cart.

## Features
- Add items to the cart
- Remove items from the cart
- Checkout functionality

## Installation
To install this project, clone the repository:

```bash
git clone https://github.com/Awanish025/Shopping_Cart.git
```

Then navigate to the project directory and install the necessary dependencies.

## Usage
Run the application and enjoy shopping!
